Visualization Narrative

Guided vs Exploratory

Guided Narrative

Visualizations used to tell a directed story or to make a point.

Think one page scrollers or visualizations embedded in articles.

Defining Characteristic

Tend to generally have a stronger focus on design and mood, as there is no cumbersome utilitarian interface to get in the way.

Exploratory Narrative

Visualizations that foster a free user-experience, letting them draw their own hypotheses and conclusions.

Usually some sort of sandbox or interface with a lot of user defined control.

Defining Characteristic

Stronger attention needs to be paid to the organization of user interface elements, to make them both visually unobstrusive and easy to learn.

Which one is your project?

Coding a world map

Topojson

To draw coordinate shapes (like country borders) using D3, we need to create a Topojson file of the borders.

Topojson files are simply .json files containing data about topology. They are a list of all of the arcs needed to draw the various borders.

Shapefiles

Unfortunately, the worldwide standard filetype for geographical borders is a .shp file.

We can't use these files with D3, but luckily we can convert them to topojson using Terminal :-)

World Shapefile

Shapefiles are fairly easy to find on the internet. Here's one I found for all of the countries in the world: world.zip

Node Package Manager (NPM)

Open Terminal and use homebrew to install the Node Package Manager. This is like a store that keeps a list of a ton of open source projects/tools (and it has the topojson converter!). 

brew install node

Topojson

Install the topojson converter! 

npm install -g topojson

Topojson

Change into the directory of the .shp file and try out a basic conversion: 

topojson -o output-file.json input-file.shp

Take a look at your .json file in your text editor!

Topojson

Let's add a better name for the list of countries. 

topojson -o output.json countries=input.shp

Topojson

Bring over all properties from the shapefile. 

topojson -o output.json countries=input.shp -p

Topojson

And pick one to use for the unique ID. 

topojson -o output.json --id-property=ISO_A2 countries=input.shp
